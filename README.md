# Text Encryption Tool

## Overview

This project implements a basic Caesar Cipher encryption and decryption tool using Python. Users can encrypt plain text or decrypt encrypted messages by specifying a shift value.

## Features

- Encrypt text
- Decrypt text
- User-defined shift value
- Easy command-line interface

## Technologies Used

- Python 3

## How to Run

```bash
python encryption.py
```
